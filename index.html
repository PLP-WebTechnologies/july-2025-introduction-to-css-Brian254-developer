CSS Syntax

CSS is written in a specific syntax that includes selectors and declarations. A declaration consists of a property and a value, separated by a colon, and ends with a semicolon.

Example of CSS

Here is an example of a simple CSS rule:

        body {
            background-color: lightblue;
            color: darkblue;
            font-family: Arial, sans-serif;
        }
